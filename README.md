# asp-core

It is a demonstration project that contains:
  1. Asp MVC 6
  2. .Net Core
  3. EntityFrameworkCore
  4. HTML5, CSS3, JQuery, Bootstrap 4 and responsive design
  5. Unit Testing using NUnit + MOQ
  6. SOLID principles
  
For simplicity some things are omitted (like strings localization, global logging, etc)
